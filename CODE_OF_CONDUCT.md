# Code of Conduct

中文摘要：这里欢迎直接、严格的工程讨论，但不接受针对人的攻击、泄露隐私或故意扰乱协作。

## Our Standard

Participants in this repository are expected to:

- stay respectful and technically grounded
- focus disagreement on behavior, code, risk, and verification
- protect private thread, project, and machine-specific information
- avoid public posting of sensitive exploit details before coordination
- keep collaboration practical and in scope

## Unacceptable Behavior

Unacceptable behavior includes:

- harassment, threats, or personal attacks
- discriminatory or hateful language
- intentional disruption of issue, review, or release work
- posting secrets, private machine data, or exploit details irresponsibly
- repeated bad-faith arguing that prevents progress

## Scope

This code of conduct applies to:

- issues
- pull requests
- discussions
- review comments
- other repository-linked collaboration spaces

## Enforcement

Maintainers may:

- request a change in behavior
- remove comments or lock threads
- reject contributions
- temporarily or permanently restrict participation for repeated or serious violations

## Reporting

Use GitHub moderation tools when appropriate. For security-sensitive content, request a private reporting path and avoid posting full exploit details publicly.

## Project Norm

This project accepts blunt technical review. It does not accept disrespectful behavior toward contributors.
